# Mini Site Builder для TravelLine Tech

Учебный проект для летней практики: мини-конструктор главной страницы портала `travelline.tech` с админ-панелью.

Публичная страница повторяет структуру оригинального сайта, а часть контента хранится в `data/data.json` и редактируется через админку.

## Стек

- Node.js
- Express
- EJS
- HTML
- CSS
- JavaScript
- JSON-файл как хранилище данных
- Multer для загрузки фото сотрудников

## Что Реализовано

Управляемые блоки:

- `Hero`: заголовок, подзаголовок, статистика.
- `Команда`: карточки сотрудников, должности, фото.
- `Клиенты`: названия клиентов и пути к логотипам.
- `Вакансии`: название, формат работы, ссылка на hh.ru.
- `Плюшки`: название бонуса и описание.

Админка позволяет редактировать данные, сохранять их в `data/data.json`, добавлять и удалять элементы списков.

## Как Запустить

1. Установить Node.js LTS: https://nodejs.org
2. Открыть папку проекта:

```bash
cd project
```

3. Установить зависимости:

```bash
npm install
```

4. Запустить сервер:

```bash
npm start
```

5. Открыть сайт в браузере:

```text
http://localhost:3000
```

6. Открыть админку:

```text
http://localhost:3000/admin.html
```

## Как Пользоваться Админкой

1. Открыть `http://localhost:3000/admin.html`.
2. Изменить нужный блок.
3. Нажать кнопку сохранения в этом блоке.
4. Открыть или обновить главную страницу `http://localhost:3000`.
5. Проверить, что изменения появились на публичной странице.

## Рендеринг

В проекте используется серверный рендеринг.

Схема работы:

```text
Браузер открывает /
Express читает data/data.json
Express передаёт данные в views/index.ejs
EJS формирует готовый HTML
Браузер получает уже готовую страницу
```

Админка работает через API:

```text
GET /api/data
POST /api/data
POST /api/upload/team-photo
```

## Структура Проекта

```text
project/
  data/
    data.json              данные управляемых блоков
  public/
    admin.html             страница админки
    css/
      admin.css            стили админки
      site-fixes.css       правки для сохранённой копии сайта
    img/                   изображения для сайта
    js/
      admin-hero.js        логика редактирования Hero
      admin-team.js        логика редактирования команды
      admin-logo.js        логика редактирования клиентов
      admin-vacation.js    логика редактирования вакансий
      admin-plushki.js     логика редактирования плюшек
    site-assets/           файлы оригинального сайта
  src/
    server.js              Express-сервер и API
  views/
    index.ejs              публичная страница с серверным рендерингом
  tests/
    storage.test.js        место для тестов
  package.json
  README.md
```

## Основные Файлы

- `src/server.js`: запускает сервер, отдаёт публичную страницу, API и статические файлы.
- `views/index.ejs`: HTML публичной страницы, куда сервер подставляет данные.
- `data/data.json`: основное хранилище контента.
- `public/admin.html`: интерфейс админки.
- `public/js/admin-*.js`: логика редактирования отдельных блоков.

## API

Получить все данные:

```text
GET /api/data
```

Сохранить все данные:

```text
POST /api/data
```

Загрузить фото сотрудника:

```text
POST /api/upload/team-photo
```
