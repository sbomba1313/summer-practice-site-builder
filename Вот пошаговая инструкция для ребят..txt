Вот пошаговая инструкция для ребят. Можно скинуть им как есть.

Что нужно один раз установить
1. Node.js (если ещё нет) — скачать с https://nodejs.org (LTS-версия), установить, перезапустить терминал. Проверка:

node --version
Должно показать версию (например v24.x). Если «команда не найдена» — перезапустить терминал/комп.

2. Git (если ещё нет) — https://git-scm.com/downloads

Поднять проект (по шагам)
Шаг 1. Получить код.

Если репозиторий ещё не склонирован:

cd C:/projects
git clone https://github.com/sbomba1313/summer-practice-site-builder.git
cd summer-practice-site-builder
Если уже склонирован — просто обновить:

git pull origin main
Шаг 2. Зайти в папку приложения. Это важно — проект лежит в подпапке project/:

cd project
Шаг 3. Установить зависимости. Эта команда прочитает package.json и скачает Express, EJS, Multer (у каждого свои, в гит они не входят):

npm install
Появится папка node_modules/ (~несколько МБ). Это нормально, она в .gitignore.

Шаг 4. Запустить сервер:

npm start
В терминале появится:

Сервер запущен на http://localhost:3000
Шаг 5. Открыть в браузере:

Публичная страница: http://localhost:3000/
Админка: http://localhost:3000/admin.html
Чтобы остановить сервер — в терминале нажать Ctrl+C.
